# ThreatLens AI

ThreatLens AI is an AI-powered cybersecurity platform that analyzes URLs, detects phishing threats, calculates risk scores, and generates automated security assessments.

## Features

- URL Validation
- URL Feature Extraction
- VirusTotal Threat Intelligence
- AI-Powered Security Analysis
- Risk Score Calculation
- Automated JSON Response

## Technologies Used

- n8n
- OpenAI API
- VirusTotal API
- JavaScript
- Webhooks

## Workflow

Webhook → URL Validation → Feature Extraction → VirusTotal Analysis → Risk Scoring → AI Assessment → Response
